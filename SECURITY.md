# Security Policy

PypBrowser is maintained by Bora.

## Reporting a Vulnerability

Please do not post security vulnerabilities publicly.

Use GitHub's private vulnerability reporting for this repository:

https://github.com/borapasa22-crypto/PypBrowser/security/advisories/new

Include:

- What happened
- Steps to reproduce it
- The operating system and PypBrowser build
- Any logs, screenshots, or proof of concept files that help confirm the issue

## Scope

Security reports should focus on PypBrowser code, packaging, branding, update metadata, browser UI behavior, or bundled configuration.

Issues inherited from upstream Firefox should also be reported to Mozilla when appropriate.

## Response

Bora will review security reports as quickly as possible and prioritize fixes based on severity, exploitability, and user impact.
