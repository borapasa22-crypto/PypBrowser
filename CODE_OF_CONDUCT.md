# PypBrowser Code of Conduct

PypBrowser is Bora's browser project, and the community around it should be practical, respectful, and focused on making the browser better.

## Standards

Good participation means:

- Be respectful when reporting bugs or giving feedback.
- Keep issues and pull requests focused on PypBrowser.
- Explain problems clearly and include screenshots, logs, or steps when useful.
- Credit other people's work when you build on it.
- Help keep the project safe, usable, and welcoming.

Unacceptable behavior includes:

- Harassment, threats, insults, or personal attacks.
- Posting private information without permission.
- Spam, scams, malware, or intentionally harmful code.
- Derailing issues with unrelated arguments.
- Repeatedly ignoring maintainer direction.

## Enforcement

Bora may edit, hide, lock, or remove comments and issues that do not fit this code of conduct. Contributors who repeatedly break these rules may be blocked from the repository.

## Reporting

Report problems through GitHub Issues if they are public project matters. For security-sensitive reports, follow [SECURITY.md](./SECURITY.md).
