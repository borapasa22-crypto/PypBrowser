# PypBrowser

PypBrowser is Bora's custom browser build: a Firefox-based project reshaped with Pyp branding, compact navigation, transparent panels, and a cleaner professional interface from the app icon to the settings screens.

## Download

Windows portable build:

- Download `downloads/windows/PypBrowser-windows-x64.zip.001`
- Download `downloads/windows/PypBrowser-windows-x64.zip.002`
- Run `downloads/windows/rebuild-windows-zip.ps1`
- Extract `PypBrowser-windows-x64.zip`
- Launch `PypBrowser.exe`

The current Windows package is about 180 MB compressed and about 302 MB extracted.

Linux and Flatpak:

- Corrected Linux metadata is in `downloads/linux/`.
- Linux binaries still need to be built on Linux and attached to a GitHub release.
- The Flatpak template is already renamed for `app.pyp_browser.pyp`.

## Built By Bora

PypBrowser is maintained, branded, packaged, and directed by Bora. The project uses Firefox as its browser-engine foundation, then turns it into a distinct PypBrowser experience with Bora's UI, naming, downloads, Linux metadata, and release identity.

## What Makes It Pyp

- Pyp logo branding across app and private browsing surfaces.
- Full-width Pyp settings with a clean left rail and larger side-to-side content.
- Transparent browser panels, downloads, library, menus, and settings styling.
- Compact mode enabled by default for a smoother, tighter browser feel.
- Windows portable build cleaned of duplicate upstream backup files.
- Pyp naming for launchers, profile paths, and Flatpak metadata.
